# 🐍 Snake Game

A classic Snake game built with Python using the built-in `turtle` graphics library. No external dependencies required.

## 🎮 How to Play

Use the arrow keys to control the snake:

- **↑ Up** — move up
- **↓ Down** — move down
- **← Left** — move left
- **→ Right** — move right

Eat the red food to grow longer and increase your score. The game ends if you hit a wall or run into yourself. Click the window to close it after game over.


## ✨ Features

- Smooth snake movement with arrow key controls
- Random food spawning
- Score tracking displayed on screen
- Wall and self-collision detection
- Game over screen with final score
