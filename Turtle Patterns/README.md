# Turtle Graphics Mini Projects

This repository contains four small projects built using Python’s turtle module.  
Each project focuses on practicing core programming concepts such as loops, randomness, movement, and color handling.

These projects were created as part of hands-on learning while improving Python fundamentals.

---

## Project Structure

This folder contains four separate mini-projects:

- Geometric Pattern Generator
- Random Walk
- Spirograph Pattern
- Spot Painting



---

## 1. Geometric Pattern Generatorgeo

This project generates repeating geometric shapes using loops and angle rotations.

Concepts practiced:

- For loops
- Shape construction (triangles, squares, polygons)
- Angle calculations
- Pattern repetition

Purpose: Understand how geometric shapes are formed mathematically and how repetition creates larger patterns.

![Geometric Pattern](images/geomatric_pattern_generator.png)


## 2. Random Walk

This project simulates a random walk where the turtle moves in random directions with random colors.

Concepts practiced:

- Random module
- Direction control
- RGB color mode
- Line thickness and speed adjustments

Purpose: Build understanding of randomness and dynamic movement in programming.

---

## 3. Spirograph Pattern

This program creates circular patterns by repeatedly drawing circles while slightly adjusting the turtle’s heading angle.

Concepts practiced:

- heading() and setheading()
- Incremental rotation
- Loop control
- Performance optimization with faster speed

Purpose: Demonstrate how small rotational changes inside a loop can create complex circular designs.

---

## 4. Spot Painting

This project recreates a grid of colored dots inspired by Damien Hirst’s spot paintings.

Concepts practiced:

- Nested loops
- Working with RGB color tuples
- Extracting colors using the colorgram library
- Grid positioning logic

Purpose: Combine structured positioning with randomness to create a balanced visual design.
