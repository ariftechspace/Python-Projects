# Turtle Racing Game

This Python project is a simple turtle race game. You can place a bet on which turtle will win, and then watch the race unfold.

## How It Works

- The program uses Python's `turtle` module to create a graphical race track.
- Six turtles of different colors are placed at the starting line.
- The user can enter a bet by choosing a color.
- Each turtle moves forward by a random distance in each step.
- The race continues until one turtle crosses the finish line.

## Controls

- A dialog box will appear asking you to enter a color to bet on before the race starts.
- No keyboard controls are required; the race progresses automatically.


## Notes

- The race uses random movements, so each game can have a different winner.
- You can modify the `colors` and `y_position` lists to add more turtles or change the race layout.
